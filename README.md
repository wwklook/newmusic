# kuwomusic
数据来源酷我音乐，UI也是参考酷我音乐网页版

通过 [http://118.126.91.79/music](http://118.126.91.79/music) 预览

ps：仅用于学习使用

## 安装
```
git clone git@github.com:wwklook/newmusic.git
npm install
```
或
```
git clone https://github.com/wwklook/newmusic.git
npm install
```

### 运行
```
npm run serve
```

### 编译
```
npm run build
```
